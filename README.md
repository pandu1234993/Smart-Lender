🏦 Smart Lender – AI Loan Approval System
📌 Project Overview
Smart Lender is a web-based Loan Approval System designed to help users check the likelihood of loan approval based on applicant details. The application provides a simple and interactive interface where users can enter loan information and receive an instant prediction result.

🎯 Objectives
Simplify the loan approval process.
Provide quick loan approval predictions.
Improve user experience with an interactive web interface.
Support single and batch loan application processing.
Demonstrate the use of AI concepts in financial applications.
✨ Features
🏦 Loan Approval Prediction
📋 Applicant Information Form
📊 Confidence Level Display
📁 Batch CSV Processing
⚡ Instant Prediction Results
📱 Responsive User Interface

▶️ How to Run the Project
Clone the repository.
git clone https://github.com/pandu1234993/Smart-Lender
Open the project folder.

Double-click index.html or open it using Live Server in Visual Studio Code.

The Smart Lender application will open in your browser.
